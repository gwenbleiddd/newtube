## 📌 Overview

NewTube is a React-based web application that allows users to search, browse, and view video content through an external API. The application provides a simplified video platform experience with a modern user interface and responsive design.

---

## 🎯 Purpose

The goal of this project is to demonstrate practical experience with React, API integration, and dynamic UI rendering by building a functional video browsing application similar to a simplified video streaming platform.

---

## ⚙️ Features

- Video search functionality
- Dynamic video listing from external API
- Embedded video playback
- Responsive user interface
- Component-based React architecture

---

## 🛠️ Tech Stack

- React (Create React App)
- JavaScript (ES6+)
- HTML5 / CSS3
- External Video API (e.g. YouTube Data API or similar)

```bash

```

---

### 2. Install dependencies
```bash

```

---

### 3. Run the application
```bash
npm start
```

Open:
http://localhost:3000
